# PlatillaTienda

Sistema de tienda online desarrollado en PHP con funcionalidades de gestión de productos, categorías y usuarios.

## Características

- Gestión de productos y categorías
- Sistema de autenticación de usuarios
- Panel de administración
- Optimización de rendimiento
- Sistema de caché
- Gestión de promociones

## Estructura del proyecto

- `app/` - Lógica de la aplicación (controladores, modelos, vistas)
- `config/` - Archivos de configuración
- `public_html/` - Archivos públicos y punto de entrada

## Instalación

1. Clona el repositorio
2. Configura la base de datos en `config/config.php`
3. Ejecuta el script de configuración inicial

## Uso

Accede al panel de administración a través de `/admin/` después de crear un usuario administrador.
